# 測試案例：財報×永續整合儀表板（Demo）

## 測試案例清單（摘要）
- TC-01 權限：一般使用者不可見待審版本
- TC-02 權限：審核者可見待審版本並可放行
- TC-03 財報：毛利率計算正確
- TC-04 財報：自由現金流計算正確
- TC-05 ESG：職災率欄位缺失時顯示不可計算
- TC-06 治理：點選資料點可回查來源與指紋
- TC-07 版本回溯：切換上一版本顯示差異與差異原因
- TC-08 資料品質：欄位漂移時自動阻擋放行並顯示告警

---

## TC-01 一般使用者不可見待審版本
前置條件：存在一筆版本狀態=待審
步驟：以一般使用者登入 → 嘗試切換到待審版本
預期結果：系統不顯示待審版本或顯示無權限提示

## TC-03 毛利率計算正確
資料：營收=100、毛利=40
步驟：進入財務趨勢頁 → 查看毛利率
預期結果：毛利率=40%

## TC-06 點選資料點可回查來源與指紋
步驟：在任一圖表點選某期資料點
預期結果：彈窗顯示公司、期間、版本號、資料源批次、指紋、更新時間

## TC-08 欄位漂移阻擋放行
前置條件：新抓取批次缺少必要欄位（例如營收）
步驟：產生待審版本 → 審核者檢視 → 嘗試放行
預期結果：系統阻擋放行並指出缺失欄位與影響指標清單
