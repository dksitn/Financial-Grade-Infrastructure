# 功能需求文件（FRD）：財報×永續整合儀表板（Demo）

## 0. 名詞定義
- 資料放行（Approval）：審核者將某次資料更新批次標記為可用
- 證據包（Evidence Pack）：該批次資料源檔案、指紋、處理紀錄、檢核結果的集合
- 版本（State Version）：可被引用的資料快照；儀表板僅讀取已放行版本

## 1. 角色與權限
- 一般使用者：可查詢儀表板、切換已放行版本、匯出
- 審核者：可檢視待審資料批次的檢核結果與證據包，執行放行/退回

## 2. 儀表板頁面與元件（Demo）
### P1 公司總覽
- 篩選器：公司代號、期間（季/年）、版本（預設最新已放行）
- 卡片：
  - 營收（季/年）
  - 淨利率
  - ROE
  - 職災率（ESG：職業安全衛生）
- 區塊：本版本資料來源摘要（財報/ESG 各自的抓取批次與更新日期）

### P2 財務趨勢
- 圖表：營收、毛利率、營益率、淨利率（可切換季/年）
- 指標定義：滑鼠提示顯示公式與來源欄位

### P3 現金流與資本效率
- 圖表：營運現金流、自由現金流（自由現金流 = 營運現金流 - 資本支出）
- 卡片：現金流量品質提示（若營收成長但現金流惡化，顯示警示）

### P4 ESG（職業安全衛生）趨勢
- 圖表：職業災害人數、職災比率、火災件數/死傷（若資料源提供）
- 說明：跨公司比較僅限同產業/同規模區間（Demo 以兩家公司做對照）

### P5 資料治理與放行（面試賣點頁）
- 區塊：目前版本狀態（已放行/待審/退回）
- 區塊：資料檢核結果（缺值、異常值、欄位漂移）
- 區塊：證據包連結（原始檔指紋、抓取時間、處理紀錄）
- 功能：切換上一版本，比對指標差異（差異需顯示原因：資料源更正/計算規則更新）

## 3. 指標計算（Demo 必備）
- 毛利率 = 毛利 / 營收
- 營益率 = 營業利益 / 營收
- 淨利率 = 稅後淨利 / 營收
- ROE = 稅後淨利 / 平均權益
- 負債比 = 總負債 / 總資產
- 流動比 = 流動資產 / 流動負債
- 自由現金流 = 營運現金流 - 資本支出

> 每個指標需在「資料源覆蓋矩陣」中列出：來源欄位、公式、更新頻率與限制。

## 4. 例外與錯誤處理
- 若 ESG 指標缺資料：顯示「資料源未揭露/本期未申報」並可切換至上一期
- 若欄位漂移：版本狀態自動轉為「待審」，禁止放行，並提示需人工處理
- 若同公司同期間出現多筆：以資料源規則（出表日期）決定最新，並留存舊值於版本差異中

## 5. 非功能需求（NFR）
- 可追溯性：任一圖表點選資料點可回查到資料源批次與指紋
- 可重現性：同一版本在任何時間查詢結果一致
- 權限隔離：未放行版本僅審核者可見
