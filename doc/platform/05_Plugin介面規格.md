# Plugin 介面規格

本文件定義 Plugin 作為「可控的查詢與計算入口」，每次執行都要回傳結果＋追溯資訊。

## 1. 共同輸入參數
- company_id
- period_range 或 year_range
- peer_group（同業集，可選）

## 2. 共同輸出結構（最小集合）
- data：表格/指標/警示
- trace：追溯資訊（使用的資料版本、公式版本、產出版本）
- dq：品質檢核摘要（通過/警告/失敗與主要 issue）

## 3. Plugin 清單（最小集合）
### 財報
- FIN_001 三表摘要
- FIN_002 財務比率
- FIN_003 異常與缺漏
- FIN_004 同業比較

### 永續
- ESG_001 指標查詢（含來源引用）
- ESG_002 強度化
- ESG_003 缺漏與口徑檢核
