# 指標層與語意層規格

本文件定義指標命名、公式版本、分母規則與展示一致性，避免「同一指標不同人算出不同結果」。

## 1. 指標命名規則
- 財報比率：FIN_RATIO_*
- 財報趨勢：FIN_TREND_*
- 永續指標：ESG_METRIC_*
- 永續強度：ESG_INTENSITY_*

## 2. 公式版本（Formula Version）
- 任一指標公式變更，必須升版（formula_version +1）
- 指標計算結果必須記錄 formula_version

## 3. 分母與口徑規則（摘要）
- 財報 ROE/ROA：優先用平均值；缺期初則用期末並標記
- 永續強度：分母必須標記來源（營收來自財報、員工數來自永續或年報）
- 缺分母：不產出該強度指標，回傳缺原因

## 4. 展示一致性（Dashboard 必遵守）
- 同頁面不混用幣別/單位
- 缺值顯示缺，不顯示 0
- 同一圖表不混用年度與季度（除非清楚分區）
