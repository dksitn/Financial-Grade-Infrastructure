# FRD｜永續分析

## S-FRD-1 功能總覽
- S-FN-01 永續總覽頁（公司層）
- S-FN-02 指標明細（年度表＋來源引用）
- S-FN-03 強度化（每營收/每人）
- S-FN-04 缺漏與口徑檢核提示
- S-FN-05 追溯資訊展示（證據包摘要）

## S-FRD-2 Plugin 對應
- ESG_001 指標查詢（含來源引用）
- ESG_002 強度化（含分母來源）
- ESG_003 缺漏與口徑檢核（缺引用/缺單位/多口徑）

## S-FRD-3 畫面/區塊（摘要）
### 永續總覽頁
- 區塊 A：排放（Scope 1/2）年度趨勢
- 區塊 B：能源/用水年度趨勢
- 區塊 C：強度化（每營收/每人）
- 區塊 D：缺漏與口徑提示
- 區塊 E：追溯入口（點任一數字）

## S-FRD-4 單位換算（MVP 必做）
- 能源：統一為 kWh（MWh 轉換為 kWh）
- 用水：統一為 m3
- 排放：統一為 tCO2e（kgCO2e 轉換為 tCO2e）
- 每次換算必記錄換算規則版本

## S-FRD-5 檢核
- S-CK-01 無來源引用 → 阻擋發佈
- S-CK-02 分母缺失 → 強度化指標不產出，顯示缺原因
- S-CK-03 多口徑 → 並列顯示並提示口徑差異
