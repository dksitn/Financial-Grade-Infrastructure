# FRD｜財報分析

## F-FRD-1 功能總覽
- F-FN-01 財報總覽頁（公司層）
- F-FN-02 三表明細（可展開）
- F-FN-03 指標與趨勢
- F-FN-04 同業比較
- F-FN-05 追溯資訊展示（證據包摘要）

## F-FRD-2 Plugin 對應
- FIN_001 三表摘要：回傳三表關鍵列＋追溯
- FIN_002 財務比率：回傳比率表＋公式版本＋追溯
- FIN_003 異常與缺漏：回傳 DQ issue 與跳動提示
- FIN_004 同業比較：回傳排名/差距（含缺值排除規則）

## F-FRD-3 畫面/區塊（摘要）
### 財報總覽頁
- 區塊 A：營收/淨利趨勢（8 季）
- 區塊 B：比率卡（毛利率/營益率/ROE/ROA/負債比）
- 區塊 C：現金流摘要（營運/投資/融資/自由現金流）
- 區塊 D：異常與缺漏提示
- 區塊 E：追溯入口（點任一數字）

### 同業比較
- 同期間同指標比較
- 缺值不排名（顯示缺）

## F-FRD-4 欄位與檢核
- F-CK-01 同頁面幣別/單位一致（否則阻擋展示並提示）
- F-CK-02 缺值顯示缺，不顯示 0
- F-CK-03 多版本預設取最新已放行版本，且可查看版本歷史
