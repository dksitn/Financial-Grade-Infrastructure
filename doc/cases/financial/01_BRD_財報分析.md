# BRD｜財報分析

## F-BRD-1 背景與問題
財報雖公開，但常見落差是：單位/幣別/期間不一致、科目映射不統一、缺漏用 0 補，導致圖表看起來正常但其實不可比較。另有稽核問題：事後找不到當時用的版本與計算口徑。

## F-BRD-2 目標
- 讓分析者能用固定口徑快速判讀最近 8 季（或 3 年）的趨勢
- 所有比率與趨勢計算有固定定義，且可追溯
- 可支援同業比較（同期間同指標，缺值不排名）

## F-BRD-3 範圍（In/Out）
### In
- 三表摘要：損益、資產負債、現金流關鍵列
- 指標：毛利率、營益率、淨利率、ROA、ROE、負債比、流動比、自由現金流
- 趨勢：YoY/QoQ、連續季變化、異常跳動提示
- 同業比較：同期間同指標排名

### Out
- 估值模型與投資建議
- 交易策略

## F-BRD-4 業務規則
- F-BR-01 缺資料顯示缺，不用 0 補
- F-BR-02 幣別/單位/期間在 Curated 層統一後才允許計算指標
- F-BR-03 ROA/ROE 分母優先用平均值；缺期初則用期末並標記
- F-BR-04 同期間多版本：保留舊版；Dashboard 預設用最新已放行版本
- F-BR-05 同業比較：缺資料不排名；不可用 0 參與排名
- F-BR-06 每次發佈必須能回推來源與版本（證據包齊備）

## F-BRD-5 需求清單（業務層）
- F-BRQ-01 顯示最近 8 季營收與淨利趨勢
- F-BRQ-02 顯示主要比率並可切換期間範圍
- F-BRQ-03 顯示現金流摘要與自由現金流
- F-BRQ-04 顯示異常跳動與缺漏提示
- F-BRQ-05 提供同業比較（自選同業或產業同業集）
- F-BRQ-06 每個指標可點開追溯：來源、版本、公式版本、核決
